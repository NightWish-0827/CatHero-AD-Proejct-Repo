# CustomToolbar

**CustomToolbar**는 Unity Editor 상단 Toolbar에 필수적인 유틸리티 기능을 통합하여, 개발 및 디버깅 워크플로우를 획기적으로 개선하기 위해 설계된 패키지입니다.

## 개요

Unity 에디터의 기본 툴바 영역에 자주 사용되는 설정과 제어 도구를 배치함으로써 작업 효율을 극대화합니다. 기존 Unity UI와의 시각적 일관성을 유지하면서도 강력한 커스터마이징 기능을 제공하는 것을 목표로 제작되었습니다.

## 주요 특징

- **실시간 제어 도구**: `Enter Play Mode` 옵션 토글, `Time Scale` 슬라이더, `Restart Current Scene` 버튼 등을 통해 에디터 제어 생산성을 높입니다.
- **Scene Switcher**: 프로젝트 내의 씬들을 자동으로 스캔하여 드롭다운 메뉴를 통해 즉각적인 씬 전환을 지원합니다.
- **성능 모니터링**: 에디터 툴바에서 실시간 FPS를 확인하고, `targetFrameRate`를 즉시 변경할 수 있습니다.
- **세부 설정 및 현지화**: `Settings` 윈도우를 통해 각 기능의 활성화 여부를 선택할 수 있으며, 한국어와 영어 다국어 지원을 포함합니다.

## 렌더링 구현 특이사항

본 패키지의 툴바 렌더링 로직은 다수의 IMGUI 하드코딩으로 구현되어 있습니다. 이는 Unity 네이티브 툴바의 내부 C++ 코드에 직접 접근할 경우 발생할 수 있는 보안 및 저작권(해적판/라이선스 우회) 이슈를 방지하고, 에디터의 안정성을 보장하기 위해 의도적으로 채택된 설계 방식입니다.
